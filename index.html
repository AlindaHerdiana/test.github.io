Karena saya berzodiak Sagitarius, tidak heran jika saya menyukai petualangan. Yap! Salah satu hobi saya adalah berpetualang atau travelling. Travelling sangatlah menyenangkan. Meskipun terkadang melelahkan, tetapi travelling dapat menghilangkan rasa stress dan jenuh kita. Kita pun dapat belajar untuk selalu beradaptasi dengan lingkungan yang baru.

Bermain Volly

Selain travelling, saya juga memiliki hobi bermain bola volly. Bola volly adalah permainan olahraga yang dimainkan oleh dua grup berlawanan. Masing-masing grup memiliki enam pemain. Saya menyukai permainan bola volly ini karena sangat menyenangkan. Selain membuat kita sehat, permainan bola volly melatih kita untuk dapat bekerja sama dalam tim. Tetapi, sekarang ini saya jarang bermain bola volly ini. Sekarang, saya bermain bola volly ini ketika libur atau ketika sedang ada acara perlombaan, contohnya seperti ketika PORSIP. Karena waktu yang tidak memungkinkan saya untuk dapat bermain volly kapan saja, maka saya bermain bola volly ini ketika libur saja.
